Project 1: One-Page Website

Make a one-page website about your favorite book. Use this opportunity to use your beginner knowledge of HTML and CSS. You can include text, images, animated gifs, reviews, characters, your personal thoughts, original images you create, etc. Be flexible with the presentation and design - you will likely not be able to do what you envision but work within those limitations. As in all projects, experiment!

Requirements:

  • One page only (index.html), a stylesheet and a folder for images and other assets.
  • Use as many HTML tags and CSS properties as you'd like.

Recommended Process:

  1. Think about what kind of content you'd like to include and start gathering it. If using text from another website, please source and provide a link.
  2. Create a local folder "project1". Folder should contain index.html, a stylesheet and a sub-folder for images and other assets.
  3. When the project is complete, upload this entire folder to your GitHub class website. The link should look something like: your-github-username.github.io/class-site/project1/. You can continue to edit your project and re-upload.
  4. Add your URL to the spreadsheet.
